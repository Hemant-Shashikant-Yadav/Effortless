Effortless Usage

You can get started with Effortless in just a few simple steps. If you're looking for more complexity, check the other tiers of usage!

Get set up

You'll need to install Effortless if you haven't yet. After that, start up Python in a new terminal or command-line.

python
MacOs Windows

Importing Effortless

After you've installed it, you'll need to import the module to access a database.

from effortless import db, Field

2. Adding Items to the Database

Use db.add() to insert new items into the database:

db.add({"name": "Alice", "age": 30}) db.add({"name":"Bob","age": 25})

3. Retrieving All Items

db.get_all() returns all items in the database:

all_items = db.get_all() print(all_items) # Output: {'1': {'name': 'Alice', 'age': 30}, '2': {'name': 'Bob', 'age': 25}}

4. Filtering Items

Use db.filter() with a Field condition to find specific items:

result = db.filter(Field("name").equals("Alice")) print(result) # Output: {'1': {'name': 'Alice', 'age': 30}}

Click the copy icon to copy the code to your clipboard.

This effortless usage allows you to quickly set up a database, add items, retrieve data, and perform basic operations without any complex setup or configuration.
